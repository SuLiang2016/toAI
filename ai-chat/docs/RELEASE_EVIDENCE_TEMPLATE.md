# AI Chat Release Evidence Template

Use one record per gate run.

## Record Fields

- App version:
- Artifact path:
- OS/profile:
- Old install state:
- New install state:
- Command used:
- Smoke result:
- Sanitized log path:
- Backup file path:
- Evidence mode: `already evidenced`, `freshly rerun`, or `deferred`
- Gate state: `STATIC PASS`, `UNPACKED SMOKE PASS`, `INSTALLER SMOKE PASS`, `BLOCKED`, or `DEFERRED`
- Owner / prerequisite note:
